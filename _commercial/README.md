# Commercial Documentation

This folder contains documentation related to the commercial/enterprise edition of the platform (formerly known as Exploop/Rediver).

These docs are kept for internal reference and are **not part of the OpenCTEM OSS project**.

## Contents

- `rfcs/` - Architecture RFCs for multi-edition split
- `deployment/` - Enterprise deployment guides
- `saas/` - SaaS billing, plans, and licensing
- `platform-agents/` - Platform-managed agent infrastructure
