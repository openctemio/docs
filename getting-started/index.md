## Prerequisites

Before you begin, ensure you have:

| Requirement | Version | Purpose |
|-------------|---------|---------|
| Docker | 20.10+ | Container runtime |
| Docker Compose | 2.0+ | Orchestration |
| Git | 2.30+ | Clone repositories |
| 8GB RAM | Minimum | Run all services |
