OpenCTEM brand — three directions

v1 (shield + iris) was honest-mediocre. v2 (open loop) was geometrically clean but anonymous. v3 explores two more characterful directions. Open this file in a browser to compare.

The question: which one feels like a brand you'd remember 3 days after seeing it once?

Direction C · v3 (letterform)
Chunky C (notched)
A chunky lowercase C with five visible notches in its curve + a marker dot in the opening. The letter IS the brand.
Strength
Strongest brand-name fusion (Stripe S, Klarna K family)
Weakness

Wordmark lockup — direction C only (the letterform fuses with wordmark)

Direction C is the only one where mark + wordmark are literally the same object: removing the C breaks the wordmark, removing the wordmark leaves a recognisable brand letter. That's the strongest possible coupling.
